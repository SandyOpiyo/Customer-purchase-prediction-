# Customer-purchase-prediction-
I built a customer purchase prediction model using logistic regression. I used age and salary as features, cleaned and scaled the data, trained the model, and achieved good accuracy. The goal was to predict customer buying behavior.
# 🧠 Customer Purchase Prediction

## 📌 Overview
This project uses machine learning to predict whether a customer will purchase a product based on their age and estimated salary. It demonstrates the end-to-end process of building a predictive model.

## 🎯 Objective
To analyze customer data and build a model that can accurately predict purchasing behavior.

## 🚀 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Dataset
The dataset contains:
- Age
- Estimated Salary
- Purchase decision (0 = No, 1 = Yes)

## ⚙️ Project Workflow
1. Data loading and exploration
2. Data cleaning and preprocessing
3. Feature selection (Age, Salary)
4. Feature scaling using StandardScaler
5. Model training using Logistic Regression
6. Model evaluation using accuracy and confusion matrix

## 📈 Results
- The model predicts whether a customer is likely to purchase
- Performance evaluated using accuracy score and confusion matrix

## 📷 Visualization
A scatter plot is used to visualize customer distribution based on age and salary.

## 💡 Key Learnings
- Importance of data preprocessing
- Feature scaling in machine learning
- Model training and evaluation techniques
